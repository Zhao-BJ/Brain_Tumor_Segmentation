# Brain Tumor Segmentation
本项目的目标是实现大脑肿瘤的分割，所使用的模型基于PyTorch实现。

## 数据集
### MICCAI BraTS 2018 Challenge
竞赛任务是分割不同神经胶质瘤子区域，包括：`1)增强肿瘤(Enhancing Tumor, ET)`、`2)肿瘤核心(Tumor Core, TC)`、`3)整个肿瘤(Whole Tumor, WT)`。
